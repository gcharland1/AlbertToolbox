# PipingEstimator

## How to Run the flask app:
### 1. Create and source a virtual environment 

- `python3 -m venv venv`
- `source venv/bin/activate`

### 2. Set up environment

- `pip install requirements.txt`
- `python run.py`

### 3. Try it!

- Navigate to `http://0.0.0.0:8080`
- Navigate to `http://0.0.0.0:8080/`
